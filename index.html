CSS Text Properties

Text-Align

Text-align determines how text is justified on the page (or in the div or other element it exists within). The text-align options include:

text-align: center;
Centers text on the page
text-align: right;
Justifies text to the right
text-align: left;
Justifies text to the left (this is the default, so you rarely need to call for it)
text-align: justify;
Spreads words evenly from right to left of the page or element that holds the text

Text-Decoration

Text-decoration has several values within it, including line, color, style, and thickness. You can set them all at once by calling text-decoration: and then setting all the values with spaces between them. Here is a run through of the different values you can set within text-decoration.

text-decoration-line:

Determines where the decoration will be relative to the text

overline;
Puts the decoration above the text
underline;
Puts the decoration below the text
line-through;
Puts the decoration through the text
underline overline;
Puts decorations above and below the text

text-decoration-color:

Sets the color of the decoration

text-decoration-style:

Sets the form the decoration line will take. This includes options like: solid, dashed, dotted, wavy, and double.

text-decoration-thickness:

Determines the boldness of the decoration line

See the Pen Text Decoration Demo by Johnson Hansen (@jfishhero) on CodePen.

Text-Transform

Text-transform has to do with capitalization of text and has 3 basic options.

text-transform: capitalize;
Capitalizes the first letter in each word, like a title
text-transform: uppercase;
Sets every letter to capital
line-text-transform: lowercase;
Sets every letter to lowercase (weird that uppercase has "capital" and "uppercase" to describe it, but I can't think of any word other than "lowercase" to describe lowercase).

See the Pen Text Transform Demo by Johnson Hansen (@jfishhero) on CodePen.

Text-Indent

Text-indent quite obviously controls how much the text is indented. It can be set in two different ways

By Length
Uses px, em, cm, etc. to set the indentation by a specific distance
By Percent
Uses a percentage of the parent element to determine how far the text should be indented

See the Pen Text Indent Demo by Johnson Hansen (@jfishhero) on CodePen.

Line-Height

Line-height determines how much space there is above and below the text. You can use three different methods to set line height

By Number
Multiplies a number by the font size to determine spacing
By Length
Uses px, em, cm, etc. to set the spacing by a specific distance
By Percent
Uses a percentage of the font size to determine the spacing
